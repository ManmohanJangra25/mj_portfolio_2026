# Manmohan Jangra — React Portfolio

The supplied HTML has been converted into a Vite + React application while preserving its original content, compiled CSS, imagery, links, and visual structure. AOS provides subtle, once-only section reveals.

Three complete, responsive case-study routes are included:

- `/case-study/b2b-platform`
- `/case-study/realtime-financial-dashboard`
- `/case-study/webflow-react-hybrid`

Each page expands the original project facts into a challenge, engineering approach, system flow, measured outcomes, and project lesson. The homepage project cards navigate directly to these routes.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

The original HTML snapshot is retained at `src/source.html` as the visual/content source of truth. Hosting analytics and platform-injected scripts are intentionally not executed by the React app.
